# m-wccup26-resource

## 项目简介

**m-wccup26-resource** 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或具体网站，仅作为静态 HTML 资源的集合与备份。

所有页面均为独立文件，不依赖外部框架或后端服务，可直接在浏览器中打开查看。

## 目录结构

```
m-wccup26-resource/
├── index.html          # 项目主页 / 导航页
├── page-01.html        # 示例页面 1
├── page-02.html        # 示例页面 2
├── assets/             # 公共资源目录（可选）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本说明文件
```

> 实际目录结构可能随归档内容更新而变化。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文档。
- 页面内容可能涉及不同主题或用途，彼此之间无强关联。
- 所有页面均以静态形式存档，便于长期保存与离线访问。
- 资源文件（如样式、脚本、图片）可能集中存放于 `assets/` 目录下，或内嵌于页面本身。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可查看。
3. 若需部署到服务器，将整个仓库目录上传至静态文件托管服务即可。
